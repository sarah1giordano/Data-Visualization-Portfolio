# About me

# Portfolio
This portfolio includes visualizations I produced as part of graduate coursework and professional experiences. It includes examples using both qualitative and quantitative data, and visuals that I produced using a variety of tools. I typically use Tableau, R, or Python to create data visualizations, and I also have some experience using photo editing and graphics software to produce descriptive visuals. Please see the descriptions of projects throughout for more details on the tools I used to produce each visual.

Visual development and production is rarely a solo journey! I have credited additional developers throughout, but please keep in mind that nearly all of the visualizations represented in this portfolio are the product of conversations, collaboration, and feedback from other parties. In some cases, I led the development of these visuals from beginning to end; in others, I picked up where others had already started. In my opinion, this collaboration and ability to translate ideas between different people and continually improve upon a visual is one of the most important steps of data visualization. Together, we create visuals that better communicate key ideas and the stories within the data.
